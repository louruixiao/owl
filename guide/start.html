# 属性

属性名类型必须描述默认值
idString组件 ID，如果不进行设置会默认生成（生成规则【组件名称-组件序号】）
displayBoolean显示隐藏true
heightString高度
widthString宽度
v-modelString | Date时间 可通过 :value.sync="datetime" 进行双向绑定，date 为变量function() { return new Date(); }
