OIcon #

使用fontawesome作为图标库，默认集成free图标库，如果需pro图标库请查看指南

属性 #

名称 描述 类型 默认值 可选值
id 组件ID String
display 显示 or 隐藏 Boolean true
icon必填 图片名称图片查询 String, Array, Object
type 图片风格 String 'fas' fas：solid 实体
fal：light 清淡的
far：regular 常规的
fad：duotone 双色调
fat：thin 细的
fab：brands 品牌标识
size 2xs 到 2xl 的 T 恤尺寸缩放以及从 1x 到 10x 的文字尺寸 String lg, xs, sm, 1x, 2x, 3x, 4x, 5x, 6x, 7x, 8x, 9x, 10x
fixed-width 将图标设置为使用固定宽度会有所帮助。 Boolean false
rotation 旋转 90 180 270读 Number 90 180 270
flip 翻转 String horizontal, vertical, both
pull right:在文字右侧，left:在文字左侧 String right, left
border 边框 Boolean false
minor 在图片组合时标注次要图标，颜色会因此翻转 Boolean false
animiate 动画 String, Array beat:心跳,fade:淡化,beat-fade:闪烁,flip:翻转,spin:旋转,spin-pulse:脉冲旋转,spin-reverse:反方向旋转
animiate-options 动画配置选项 Object
animiate-scale 在beat 和 beat-fade 动画中设置图标将缩放的最大值 Number
animiate-opacity 在fade 和 beat-fade 动画中设置图标将淡入淡出的最低不透明度值 Number
animiate-flip-axis 在flip动画中设置旋转轴 String, Array 'x' , 'y' , 'z'
animiate-flip-angle 在flip动画中设置旋转角度 Number
